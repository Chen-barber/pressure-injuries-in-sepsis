## 步骤1：安装Git LFS（用于大文件）

你的.pkl文件很大（约35MB），需要使用Git LFS：

```powershell
winget install --id Git.Git-LFS -e --source winget
```

**安装后，关闭并重新打开PowerShell！**

然后验证：
```powershell
git lfs version
```

---

## 步骤2：进入项目目录并初始化

```powershell
cd F:\Data_Analysis\meeting
git init
```

---

## 步骤3：配置Git LFS并上传代码

```powershell
# 初始化Git LFS
git lfs install

# 跟踪.pkl大文件
git lfs track "*.pkl"

# 添加.gitattributes文件
git add .gitattributes

# 添加所有文件
git add .

# 提交
git commit -m "Initial commit - Pressure Injuries in Sepsis Calculator"

# 连接到你的GitHub仓库
git remote add origin https://github.com/Chen-barber/pressure-injuries-in-sepsis.git

# 设置主分支
git branch -M main

# 推送到GitHub
git push -u origin main
```

---

## 步骤4：创建Personal Access Token

推送时会要求输入密码，GitHub不再支持密码登录，需要使用Token：

1. **访问：** https://github.com/settings/tokens
2. **点击：** "Generate new token" → "Generate new token (classic)"
3. **设置：**
   - Note: `Deploy Pressure Injuries in Sepsis`
   - Expiration: 选择期限（建议90天或No expiration）
   - **勾选权限：** `repo`（全部权限）
4. **点击：** "Generate token"
5. **立即复制token**（类似：`ghp_xxxxxxxxxxxxxxxxxxxx`），只显示一次！
6. **推送时使用：**
   - Username: `Chen-barber`
   - Password: **粘贴刚才复制的token**

## 步骤6：部署到Railway

1. **访问：** https://railway.app
2. **点击：** "Login" → "Login with GitHub"
3. **授权：** 允许Railway访问你的GitHub
4. **创建项目：**
   - 点击 "New Project"
   - 选择 "Deploy from GitHub repo"
   - 找到并选择：`Chen-barber/pressure-injuries-in-sepsis`
5. **等待部署：**
   - Railway会自动检测Dockerfile
   - 开始构建（约3-5分钟）
6. **获取链接：**
   - 部署完成后，Settings → Domains
   - 会看到一个URL，这就是你的网站链接！
